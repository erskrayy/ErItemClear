# 🧹 ErItemClear

**ErItemClear** — это гибкий и оптимизированный плагин для управления лежащими на земле предметами (дропом) на вашем сервере Minecraft (Paper/Spigot 1.21). Плагин позволяет детально настраивать время жизни предметов, добавлять информативные голограммы и применять визуальные эффекты при их удалении.

## ✨ Основные возможности

* **Голограммы таймера:** Кастомные голограммы над лежащими предметами, показывающие оставшееся время до удаления и количество предметов в стаке (с поддержкой HEX-цветов).
* **Кастомное время по материалам:** Настраивайте разное время удаления для разных блоков и предметов (например, земля удаляется быстро, а алмазы лежат дольше).
* **Разные миры — разные правила:** Возможность задать индивидуальное базовое время очистки для обычного мира, Незера и Энда.
* **Исключения и фильтры:**
  * Белый и чёрный списки (Whitelist/Blacklist) материалов, которые не должны удаляться.
  * **Поддержка NBT-тегов:** Игнорирование кастомных предметов из других плагинов (ItemsAdder, Oraxen и др.) по ключам `namespace:key`.
* **Визуальные эффекты:** Воспроизведение звуков и частиц (Particles) в момент исчезновения предмета.

## ⚙️ Команды

| Команда | Описание |
| :--- | :--- |
| `/eritemclear reload` | Перезагружает конфигурацию плагина без перезапуска сервера. |
| `/eritemclear clear` | Принудительно очищает все предметы, отслеживаемые плагином. |
| `/eritemclear info` | Показывает подробную статистику по предметам (общее количество и топ по материалам). |

*Разрешение для всех команд:* `eritemclear.admin` (или наличие прав оператора).

## 📄 Конфигурация (config.yml)

Ниже представлен стандартный файл конфигурации. Вы можете гибко настроить каждую деталь под нужды вашего сервера.

```yaml
# ===== ОСНОВНЫЕ НАСТРОЙКИ =====
# Стандартное время удаления предметов (в секундах)
default-time: 60

# Формат голограммы (поддерживает HEX цвета, например #FB9C08)
# {time} - оставшееся время
# {count} - количество предметов в стаке
hologram-format: "#FB9C08🔥 &n{time}&r сек &7(x{count})"

# ===== КАСТОМНЫЕ ВРЕМЕНА ПО МАТЕРИАЛАМ =====
clear-custom:
  enabled: true
  materials:
  # DIAMOND_BLOCK: 180
  # NETHERITE_SWORD: 300
  # DIRT: 30

# ===== РАЗНЫЕ ВРЕМЕНА ДЛЯ РАЗНЫХ МИРОВ =====
world-times:
  enabled: false
  worlds:
    world: 60
    # world_nether: 45
    # world_the_end: 90

# ===== ИСКЛЮЧЕНИЯ (ПРЕДМЕТЫ, КОТОРЫЕ НЕ УДАЛЯТЬ) =====
exclusions:
  enabled: false
  whitelist-mode: false
  materials:
  # - DIAMOND
  # - EMERALD

# ===== ИСКЛЮЧЕНИЯ ПО NBT-ТЕГУ =====
nbt-exclusions:
  enabled: false
  keys:
  # - "customblocks:block_id"
  # - "itemsadder:custom_item"

# ===== ВИЗУАЛЬНЫЕ ЭФФЕКТЫ ПРИ УДАЛЕНИИ =====
effects:
  enabled: false
  effect-type: "PARTICLE" # PARTICLE, SOUND, BOTH
  particle: "CLOUD"
  sound: "ENTITY_ITEM_PICKUP"
  volume: 1.0
  pitch: 1.0

# ===== СООБЩЕНИЯ ПЛАГИНА =====
messages:
  prefix: "&8[&bErItemClear&8] "
  no-permission: "&cУ вас нет прав!"
  reload-success: "&aКонфигурация успешно перезагружена!"
  clear-success: "&aОчищено предметов: &f{count}"
  usage: "&fИспользование: &a/eritemclear reload &7| &a/eritemclear clear &7| &a/eritemclear info"
  info-header: "&8=== &bErItemClear Info &8==="
  info-total: "&fВсего в трекинге: &a{total} &fпредметов"
  info-material: "&f{material}&8: &a{count}"
  info-footer: "&8===================="
