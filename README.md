# 🗑️ ErItemClear

Умный и производительный плагин для управления лежащими на земле предметами (дропом) на серверах Minecraft. Позволяет детально настраивать время жизни предметов, добавляет информативные голограммы с таймером и красивые визуальные эффекты при их удалении.

Разработано специально для современных ядер **Paper/Spigot**.

## ✨ Главные особенности

* **Гибкие таймеры очистки:**
  * Установка базового времени удаления для всего сервера.
  * Индивидуальные таймеры для конкретных материалов (например, земля удаляется быстро, а незеритовый меч — долго).
  * Настройка времени жизни предметов в зависимости от мира (world, world_nether и т.д.).
* **Информативные голограммы:** Показывают точное оставшееся время и количество предметов в стаке. Полностью поддерживают HEX-цвета и стандартные цветовые коды.
* **Продвинутая система исключений:**
  * **По материалу:** Режимы Blacklist (удалять всё, кроме списка) и Whitelist (удалять только то, что в списке).
  * **По NBT-тегам:** Надежная защита кастомных предметов из других плагинов (ItemsAdder, Oraxen, MMOItems и т.д.) от случайного удаления. Достаточно указать ключ в формате `namespace:key`.
* **Визуальные эффекты:** Воспроизведение звуков и частиц (Particle) в момент очистки предмета по таймеру.
* **Кастомизация:** 100% сообщений плагина можно изменить под стилистику вашего проекта.

## ⚙️ Системные требования

* **Ядро:** Paper, Spigot или их форки.
* **Версия Java:** 17 – 21.

## 🚀 Команды и Права

Для использования всех команд требуется наличие прав оператора (OP) или соответствующего пермишена (по умолчанию `eritemclear.admin`):

* `/eritemclear reload` (алиас: `/eic reload`) — Моментально перезагрузить файл конфигурации без рестарта сервера.
* `/eritemclear clear` (алиас: `/eic clear`) — Принудительно очистить все отслеживаемые плагином предметы в данный момент.
* `/eritemclear info` (алиас: `/eic info`) — Вывести подробную статистику по лежащему на сервере дропу (общее количество и разбивка по материалам).

## 🛠️ Установка

1. Скачайте скомпилированный `.jar` файл плагина из раздела **Releases**.
2. Поместите скачанный файл в папку `plugins` вашего сервера.
3. Запустите или перезапустите сервер.
4. Перейдите в папку `plugins/ErItemClear`, настройте файл `config.yml` под нужды вашей экономики.
5. Введите в консоль или в игру команду `/eic reload` для применения новых настроек.
